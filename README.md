# Homebrew Tap

Homebrew formulae for Crazytieguy's projects.

## Usage

```bash
brew tap Crazytieguy/tap
brew install <formula>
```

## Available Formulae

- **live-trans** - Real-time translation of Spotify song lyrics

## Direct Installation

You can also install without tapping:

```bash
brew install Crazytieguy/tap/<formula>
```
